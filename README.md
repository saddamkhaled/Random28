# Random28
Random 28 Luck game ! 

RANDOM 28:

*you have to choose 7 numbers from 1 to 28. 

*you have 5 bonus numbers.

*once you get 4 correct numbers you win!

![alt text](https://i.imgur.com/dpauAKT.png)

## Requirements
Python3
## Usage
Linux

`Python3 Random 28.py`

Windows

`Python Random 28.py`
